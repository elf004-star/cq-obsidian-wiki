## 第一章 总则

**第一条** 为了促进社会主义市场经济健康发展，鼓励和保护公平竞争，预防和制止不正当竞争行为，保护经营者和消费者的合法权益，制定本法。

**第二条** 经营者在生产、经营活动中，应当遵循自愿、平等、公平、诚信的原则，遵守法律和商业道德，公平参与市场竞争。

---

## 第二章 不正当竞争行为

### 第一节 混淆行为（第七条）

经营者不得实施下列混淆行为，引人误认为是他人商品或与他人存在特定联系：

1. 擅自使用与他人**有一定影响的商品名称、包装、装潢**等相同或近似的标识
2. 擅自使用他人**有一定影响的名称**（包括简称、字号等）、**姓名**（包括笔名、艺名、网名、译名等）
3. 擅自使用他人**有一定影响的域名主体部分、网站名称、网页、新媒体账号名称、应用程序名称或图标**
4. 其他足以引人误认的混淆行为

> 将他人注册商标、未注册的驰名商标作为企业名称字号使用，或设为搜索关键词的，也属于混淆行为。

### 第二节 商业贿赂（第八条）

经营者不得给予财物或其他手段贿赂：

| 受贿主体 | 说明 |
|---------|------|
| 交易相对方的工作人员 | — |
| 受交易相对方委托办理相关事务的单位或个人 | — |
| 利用职权或影响力影响交易的单位或个人 | — |

**合规要点：**
- 交易活动中可以**明示方式**支付折扣或佣金
- 必须**如实入账**
- 接受折扣、佣金的经营者也应当如实入账
- 工作人员行贿的，认定为经营者行为（除非有证据证明无关）

### 第三节 虚假宣传（第九条）

经营者禁止行为：
- 对商品**性能、功能、质量、销售状况、用户评价、曾获荣誉**等作虚假或引人误解的商业宣传
- 通过**组织虚假交易、虚假评价**等方式帮助其他经营者进行虚假宣传

### 第四节 侵犯商业秘密（第十条）

**侵犯行为类型：**
1. 以盗窃、贿赂、欺诈、胁迫、电子侵入或其他不正当手段获取
2. 披露、使用或允许他人使用上述手段获取的商业秘密
3. 违反保密义务，披露、使用或允许他人使用所掌握的商业秘密
4. 教唆、引诱、帮助他人违反保密义务获取、披露、使用商业秘密

> **商业秘密定义：** 不为公众所知悉、具有商业价值并经权利人采取相应保密措施的技术信息、经营信息等商业信息。

### 第五节 有奖销售禁止（第十一条）

| 禁止情形 |
|---------|
| 奖的种类、兑奖条件、奖金金额或奖品等**信息不明确** |
| 无正当理由**变更**有奖销售信息 |
| 谎称有奖或**内定人员中奖**等欺骗方式 |
| 抽奖式有奖销售**最高奖超过五万元** |

### 第六节 商业诋毁（第十二条）

经营者不得**编造、传播或指使他人编造、传播虚假信息或误导性信息**，损害其他经营者的商业信誉、商品声誉。

### 第七节 网络不正当竞争（第十三条）

**互联网专条规定：**

禁止利用数据、算法、技术、平台规则等实施：
- 未经同意**插入链接、强制跳转**
- **误导、欺骗、强迫**用户修改、关闭、卸载其他经营者的产品或服务
- **恶意不兼容**
- 其他妨碍、破坏其他经营者网络产品或服务正常运行的行为

**数据侵权：**
- 不得以欺诈、胁迫、破坏技术措施等不正当方式获取、使用其他经营者的数据

**平台规制：**
- 不得滥用平台规则实施**虚假交易、虚假评价或恶意退货**

### 第八节 强制交易（第十四条）

平台经营者不得**强制或变相强制**平台内经营者以**低于成本的价格**销售商品。

### 第九节 优势地位滥用（第十五条）

大型企业等不得滥用资金、技术、交易渠道、行业影响力等优势地位，要求中小企业接受：
- 明显不合理的付款期限
- 明显不合理的付款方式
- 明显不合理的条件和违约责任

禁止拖欠中小企业的货物、工程、服务账款。

## 第三章 监督检查

### 调查措施（第十六条）

| 措施类型 | 说明 |
|---------|------|
| 场所检查 | 进入涉嫌不正当竞争行为的经营场所 |
| 询问调查 | 询问经营者、利害关系人及有关单位、个人 |
| 资料查询 | 查询、复制相关协议、账簿、单据、文件、记录等 |
| 查封扣押 | 查封、扣押与涉嫌行为有关的财物 |
| 账户查询 | 查询涉嫌不正当竞争行为的经营者的银行账户 |

> **注意：** 采取查封、扣押、账户查询措施需向**设区的市级以上**监督检查部门主要负责人书面报告并经批准。

### 约谈与举报

- **约谈制度（第十八条）：** 可对涉嫌违法经营者进行约谈
- **保密义务（第十九条）：** 对调查过程中知悉的商业秘密、个人隐私和个人信息负有保密义务
- **举报制度（第二十条）：** 任何单位和个人有权举报，监督检查部门应向社会公开受理方式，为举报人保密

### 平台责任（第二十一条）

平台经营者应当：
1. 在平台服务协议和交易规则中**明确平台内公平竞争规则**
2. 建立**不正当竞争举报投诉和纠纷处置机制**
3. 发现平台内经营者实施不正当竞争行为，应**及时依法采取必要处置措施**
4. 保存有关记录，并向**平台经营者住所地县级以上**监督检查部门报告

## 第四章 法律责任

### 民事责任（第二十二条）

**赔偿数额确定顺序：**
1. 实际损失
2. 侵权所得利益
3. 故意侵犯商业秘密（情节严重）可获 **1-5倍** 赔偿

> **法定赔偿上限：** 权利人实际损失、侵权人获益难以确定的，由法院判决给予**500万元以下**赔偿。

### 行政责任概览

| 违法行为 | 罚款幅度 |
|---------|---------|
| 混淆行为 | 违法经营额5倍以下；无/不足5万则25万以下；情节严重吊销执照 |
| 商业贿赂 | 10万-100万；情节严重100万-500万，可吊销执照 |
| 虚假宣传 | 100万以下；情节严重100万-200万，可吊销执照 |
| 侵犯商业秘密 | 10万-100万；情节严重100万-500万 |
| 有奖销售违规 | 5万-50万 |
| 商业诋毁 | 10万-100万；情节严重100万-500万 |
| 网络不正当竞争 | 10万-100万；情节严重100万-500万 |
| 强制低价销售 | 5万-50万；情节严重50万-200万 |
| 优势地位滥用 | 100万以下；情节严重100万-500万 |

### 从轻减轻情节（第三十二条）

经营者有**主动消除或减轻违法行为危害后果**等法定情形的，依法从轻或减轻行政处罚；违法行为轻微并及时纠正，没有造成危害后果的，**不予行政处罚**。

### 信用记录（第三十三条）

受到行政处罚的，由监督检查部门**记入信用记录**，依照有关法律、行政法规的规定予以公示。

### 妨碍调查责任（第三十五条）

- 对个人：**1万元以下**罚款
- 对单位：**10万元以下**罚款

### 商业秘密民事诉讼举证责任（第三十九条）

**权利人提供初步证据后，涉嫌侵权人应当证明：**
- 权利人所主张的商业秘密不属于本法规定的商业秘密
- 其不存在侵犯商业秘密的行为

## 第五章 附则

### 域外适用（第四十条）

> 在**中华人民共和国境外**实施本法规定的不正当竞争行为，扰乱境内市场竞争秩序，损害境内经营者或消费者合法权益的，**依照本法处理**。

## 核心要点速览

| 章节 | 关键内容 |
|-----|---------|
| **混淆行为** | 保护有一定影响的商品名称、包装装潢、名称、姓名、域名、网站名称等 |
| **商业贿赂** | 禁止向交易相对方工作人员、利用职权者行贿；折扣佣金须如实入账 |
| **虚假宣传** | 禁止虚假商业宣传及组织虚假交易、虚假评价 |
| **商业秘密** | 明确定义；举证责任倒置 |
| **有奖销售** | 最高奖不得超过5万元 |
| **网络竞争** | 专门规定互联网不正当竞争行为类型（2017年新增，2025年扩充） |
| **平台责任** | 建立公平竞争规则、举报机制、及时处置并报告 |
| **优势地位滥用** | 禁止大企业拖欠中小企业账款（2025年新增） |
| **法律责任** | 最高500万元罚款；情节严重可吊销执照 |
| **域外适用** | 境外不正当竞争行为扰乱境内秩序的适用本法 |
