## 第一章：总则

**第一条** 为了加强土地管理，维护土地的社会主义公有制，保护、开发土地资源，合理利用土地，切实保护耕地，促进社会经济的可持续发展，根据宪法，制定本法。

**基本国策：**> "十分珍惜、合理利用土地和切实保护耕地是我国的基本国策"

**土地分类：**
- **农用地**：耕地、林地、草地、农田水利用地、养殖水面等
- **建设用地**：城乡住宅、公共设施、工矿、交通水利、旅游、军事设施用地等
- **未利用地**：农用地和建设用地以外的土地

**土地公有制：** 中华人民共和国实行土地的社会主义公有制，即全民所有制和劳动群众集体所有制。土地的社会主义公有制，即全民所有制和劳动群众集体所有制。

全民所有，即国家所有土地的所有权由**国务院**代表国家行使。

任何单位和个人不得侵占、买卖或者以其他形式非法转让土地。土地使用权可以依法转让。

国家为了公共利益的需要，可以依法对土地实行征收或者征用并给予补偿。

国家依法实行国有土地有偿使用制度。但是，国家在法律规定的范围内划拨国有土地使用权的除外。

**土地用途管制：** 国家编制土地利用总体规划，规定土地用途，将土地分为农用地、建设用地和未利用地。严格限制农用地转为建设用地，控制建设用地总量，对耕地实行特殊保护。

**主管部门：** 国务院自然资源主管部门统一负责全国土地的管理和监督工作。

## 第二章：土地所有权和使用权

**城市土地：** 城市市区的土地属于国家所有。

**农村土地：** 农村和城市郊区的土地，除由法律规定属于国家所有的以外，属于农民集体所有；宅基地和自留地、自留山，属于农民集体所有。

**土地承包：**
- 家庭承包耕地的承包期为**三十年**，届满再延长三十年
- 草地承包期为**三十年至五十年**，届满依法相应延长
- 林地承包期为**三十年至七十年**，届满依法相应延长

**争议处理：** 当事人协商→人民政府处理→30日内可向法院起诉。在土地所有权和使用权争议解决前，任何一方不得改变土地利用现状。

## 第三章：土地利用总体规划

### 规划原则

1. 落实国土空间开发保护要求，严格土地用途管制
2. 严格保护**永久基本农田**，严控非农建设占用农用地
3. 提高土地节约集约利用水平
4. 统筹安排城乡生产、生活、生态用地
5. 保护和改善生态环境
6. 占用耕地与开发复垦耕地**数量平衡、质量相当**

### 国土空间规划

> "已经编制国土空间规划的，不再编制土地利用总体规划和城乡规划。"

### 审批制度

| 规划类型 | 审批机关 |
|----------|----------|
| 省、自治区、直辖市规划 | 报**国务院**批准 |
| 省会城市、百万人口以上城市 | 经省级审查后报**国务院**批准 |
| 其他规划 | 逐级上报省级批准 |

## 第四章：耕地保护（重点）

### 占补平衡制度

- 原则："**占多少，垦多少**"
- 无条件开垦的：缴纳**耕地开垦费**，专款用于开垦新耕地

### 永久基本农田保护

**应划为永久基本农田的耕地：**
- 重要农产品（粮、棉、油、糖）生产基地内耕地
- 有良好水利水保设施的耕地及高标准农田
- 蔬菜生产基地
- 农业科研、教学试验田

> "各省、自治区、直辖市划定的永久基本农田一般应当占本行政区域内耕地的**百分之八十以上**"

- 占用永久基本农田必须经**国务院批准**
- **禁止**擅自调整规划规避审批

### 禁止行为

- 禁止占用耕地建窑、建坟、擅自建房、挖砂、采石、采矿、取土
- 禁止占用永久基本农田发展**林果业和挖塘养鱼**
- 禁止毁坏森林、草原开垦耕地
- 禁止围湖造田和侵占江河滩地
- 闲置耕地超**1年**须缴闲置费；连续**2年**未使用可无偿收回使用权

## 第五章：建设用地

### 征地审批权限

| 情形 | 审批机关 |
|------|---------|
| 永久基本农田 | **国务院** |
| 永久基本农田外耕地超35公顷 | **国务院** |
| 其他土地超70公顷 | **国务院** |
| 上述以外土地 | 省、自治区、直辖市人民政府 |

### 征地程序

1. 开展现状调查和**社会稳定风险评估**
2. 公告至少**30日**，听取相关方意见
3. 多数成员认为方案不符规定的，须**召开听证会**
4. 签订补偿安置协议
5. 申请征收土地

### 征地补偿原则

> "征收土地应当给予**公平、合理的补偿**，保障被征地农民原有生活水平不降低、长远生计有保障。"

补偿内容包括：土地补偿费、安置补助费、农村村民住宅补偿、地上附着物和青苗补偿、社会保障费用。

区片综合地价**至少每三年**调整或重新公布一次。

农村村民住宅补偿原则：**先补偿后搬迁、居住条件有改善**。

### 国有土地有偿使用

新增建设用地土地有偿使用费：**30%上缴中央财政，70%留地方**。

### 宅基地规定

- **一户只能拥有一处宅基地**
- 出卖、出租、赠与住宅后，再申请宅基地**不予批准**
- 国家允许进城落户村民**依法自愿有偿退出**宅基地

### 集体经营性建设用地

- 符合规划、依法登记的集体经营性建设用地可**出让、出租**
- 须经本集体经济组织成员村民会议**三分之二以上**成员同意
- 取得的使用权可转让、互换、出资、赠与或抵押

---

## 第六章：监督检查

监督检查部门权力：
- 查阅、复制土地权利文件
- 要求当事人作出说明
- 进入现场勘测
- 责令停止违法行为
- 发现犯罪行为须移送司法机关

---

## 第七章：法律责任

| 违法行为 | 主要法律后果 |
|---------|------------|
| 非法转让土地 | 没收违法所得、限期拆除或没收建筑物、罚款、追刑责 |
| 非法占用耕地建窑建坟等 | 责令限期改正或治理、罚款、追刑责 |
| 拒不履行土地复垦义务 | 限期改正→缴纳复垦费→罚款 |
| 非法占用土地 | 退还土地、限期拆除、没收建筑物、罚款、追刑责 |
| 非法批准用地 | 批准文件**无效**、处分责任人、追刑责、赔偿损失 |
| 侵占挪用征地补偿费 | 构成犯罪追刑责，否则给予处分 |
| 对责令拆除不服 | **15日内**向法院起诉；期满不起诉不拆除的，申请法院强制执行 |

---

## 第八章：附则

- **外商投资企业**使用土地适用本法（法律另有规定除外）
- 编制国土空间规划前，已批准的土地利用总体规划和城乡规划**继续执行**
- **施行日期**：1999年1月1日起施行（2004年修正后重新公布施行）

## 2019年修正核心变化

| 变化领域 | 主要内容 |
|----------|----------|
| **集体土地入市** | 集体经营性建设用地可依法出让、出租，不再必须先征为国有 |
| **宅基地自愿有偿退出** | 允许进城落户农民依法自愿有偿退出宅基地 |
| **征地程序** | 新增社会稳定风险评估、听证会要求 |
| **征地补偿** | 原则改为"公平合理补偿，保障原有生活水平不降低" |
| **永久基本农田** | 明确永久基本农田特殊保护地位 |
| **土地督察** | 新增国务院授权机构对省级政府进行督察 |
