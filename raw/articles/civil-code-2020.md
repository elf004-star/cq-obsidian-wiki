## 第一编 总则

### 基本规定（第一条至第十二条）

**立法目的与调整范围**
- 保护民事主体的合法权益，调整民事关系
- 适应中国特色社会主义发展要求，弘扬社会主义核心价值观

> **核心原则**：平等、自愿、公平、诚信、守法、绿色

### 自然人（第十三条至第五十六条）

**民事权利能力** 从出生时起到死亡时止，民事权利能力一律平等

**民事行为能力划分**

| 年龄/状况 | 类别 | 行为效力 |
|-----------|------|----------|
| 18周岁以上 | 完全民事行为能力人 | 可独立实施民事法律行为 |
| 16-18周岁以自己劳动收入为主要生活来源 | 视为完全民事行为能力人 | 可独立实施民事法律行为 |
| 8-18周岁 | 限制民事行为能力人 | 纯获利益或与年龄智力相适应有效；其他需法定代理人同意或追认 |
| 不满8周岁 | 无民事行为能力人 | 由法定代理人代理 |
| 不能辨认/不能完全辨认自己行为 | 无/限制民事行为能力人 | 由法定代理人代理 |

**监护制度**
- 父母对未成年子女：**抚养、教育和保护**的义务
- 成年子女对父母：**赡养、扶助和保护**的义务
- **成年人意定监护**：具有完全民事行为能力的成年人可事先协商确定监护人
- **监护人职责**：代理被监护人实施民事法律行为，保护人身权利、财产权利等

**宣告失踪与宣告死亡**

| 情形 | 时间要求 |
|------|----------|
| 宣告失踪 | 下落不明满2年 |
| 宣告死亡 | 下落不明满4年；或因意外事件满2年 |
| 意外事件证明不可能生存 | 可不受2年限制 |

### 法人（第五十七条至第一百零一条）

**法人分类**

| 类型 | 举例 |
|------|------|
| 营利法人 | 有限责任公司、股份有限公司 |
| 非营利法人 | 事业单位、社会团体、基金会、社会服务机构 |
| 特别法人 | 机关法人、农村集体经济组织、城镇农村合作经济组织、基层群众性自治组织 |

**法人责任**：法人以其全部财产独立承担民事责任

### 民事权利（第一百零九条至第一百三十二条）

**自然人权利**：人身自由、人格尊严、生命权、身体权、健康权、姓名权、肖像权、名誉权、荣誉权、隐私权、婚姻自主权

**知识产权范围**：作品；发明、实用新型、外观设计；商标；地理标志；商业秘密；集成电路布图设计；植物新品种等

### 民事法律行为（第一百三十三条至第一百六十条）

**有效条件**：行为人具有相应的民事行为能力；意思表示真实；不违反法律、行政法规的强制性规定，不违背公序良俗

**无效/可撤销情形**：
- 无民事行为能力人实施 → 无效
- 重大误解、欺诈、胁迫、显失公平 → 可撤销
- 虚假意思表示 → 无效

### 代理（第一百六十一条至第一百七十五条）

- **委托代理**与**法定代理**
- **无权代理**与**表见代理**（相对人有理由相信代理权）
- **紧急救助免责**：因自愿实施紧急救助行为造成受助人损害的，救助人不承担民事责任

### 民事责任（第一百七十六条至第一百八十七条）

**承担方式**：停止侵害、排除妨碍、消除危险、返还财产、恢复原状、修理重作更换、继续履行、赔偿损失、支付违约金、消除影响恢复名誉、赔礼道歉

**见义勇为补偿**：保护他人民事权益使自己受损害的，由侵权人承担侵权责任，受益人可给予适当补偿

### 诉讼时效（第一百八十八条至第一百九十九条）

- **普通诉讼时效**：3年（自权利人知道或应当知道权利受损及义务人之日起）
- **最长保护期**：20年
- **特殊时效**：未成年人遭受性侵害，自年满18周岁起算

## 第二编 物权

### 通则

**物权定义**：权利人依法对特定的物享有直接支配和排他的权利，包括**所有权、用益物权、担保物权**

**公示原则**：
- 不动产物权：**登记**生效
- 动产物权：**交付**生效
- 船舶、航空器、机动车等：未经登记**不得对抗善意第三人**

### 所有权

**国家、集体、私人物权平等保护**

**业主共同决定事项**（需2/3以上业主参与表决）：

| 事项 | 表决要求 |
|------|----------|
| 制定/修改议事规则、管理规约 | 过半数同意 |
| 选举/更换业主委员会 | 过半数同意 |
| 选聘/解聘物业服务企业 | 过半数同意 |
| 使用维修资金 | 过半数同意 |
| **筹集维修资金** | 3/4以上同意 |
| **改建、重建建筑物** | 3/4以上同意 |
| **改变共有部分用途/利用共有部分经营** | 3/4以上同意 |

### 用益物权

**土地承包经营权**：耕地承包期**30年**；草地**30-50年**；林地**30-70年**

**建设用地使用权**
- 可在**地表、地上、地下**分别设立
- 住宅建设用地使用权届满：**自动续期**

**居住权**（新增）
- 无偿设立（另有约定除外）
- **不得转让、继承**
- **不得出租**（另有约定除外）

### 担保物权

**抵押权**
- 可抵押财产：建筑物、建设用地使用权、海域使用权、生产设备、原材料、半成品、产品、船舶、航空器、交通运输工具等
- **不得抵押**：土地所有权；宅基地使用权；公益设施；所有权不明/争议财产；查封扣押财产

**质权**：动产质权自**交付**时设立；权利质权：票据、债券、股票、知识产权、应收账款等

**留置权**：同一动产已设抵押权或质权又被留置的，**留置权人优先受偿**

## 第三编 合同

### 合同通则

**合同订立**：形式：书面、口头或其他形式；数据电文视为书面形式；**格式条款**：提供方应遵循公平原则，采取合理方式提示对方注意

**合同履行原则**：全面履行；诚信原则（通知、协助、保密等义务）；绿色原则

**债权债务转让**：债权转让**通知债务人**生效；债务转移须经**债权人同意**

### 典型合同

| 合同类型 | 关键规定 |
|----------|----------|
| **买卖合同** | 风险转移：交付前归出卖人，交付后归买受人 |
| **赠与合同** | 权利转移前可撤销；公证/公益/道德义务性质不可撤销 |
| **借款合同** | 禁止高利放贷；利息不得预先扣除 |
| **保证合同** | **一般保证**与**连带责任保证**；未约定按一般保证处理 |
| **租赁合同** | 期限**不超过20年**；6个月以上应采用书面形式 |
| **物业服务合同** | 物业费不得停水停电催交 |
| **保理合同** | 有追索权/无追索权保理 |
| **建设工程合同** | 承包人工程价款**优先受偿** |

### 准合同

**无因管理**：无法定或约定义务，为避免他人利益受损管理事务。可请求偿还必要费用；受损可请求适当补偿。

**不当得利**：无法律根据取得利益 → 返还；不知情且利益不存在 → 不承担返还义务；知情 → 返还并赔偿损失。

## 第四编 人格权

**独立成编**（民法典重要创新）

### 具体人格权

| 权利 | 核心内容 |
|------|----------|
| **生命权、身体权、健康权** | 禁止买卖人体细胞、组织、器官、遗体；禁止性骚扰 |
| **姓名权/名称权** | 决定、使用、变更或许可他人使用；不得干涉、盗用、假冒 |
| **肖像权** | 未经同意不得制作、使用、公开；合理使用情形除外 |
| **名誉权** | 禁止侮辱、诽谤；新闻报道可合理使用 |
| **隐私权** | 保护私人生活安宁和私密空间、活动、信息 |

**个人信息保护**
- 定义：可识别自然人的各种信息
- 处理原则：**合法、正当、必要**
- 处理条件：同意；公开规则；明示目的、方式、范围

## 第五编 婚姻家庭

### 基本原则

- **婚姻自由**、**一夫一妻**、**男女平等**
- 禁止包办买卖婚姻、借婚姻索取财物
- 禁止重婚、有配偶者与他人同居、家庭暴力、虐待遗弃

### 结婚

- **法定婚龄**：男22周岁、女20周岁
- **禁止结婚**：直系血亲、三代以内旁系血亲
- **无效婚姻**：重婚、禁止结婚的亲属关系、未到法定婚龄
- **可撤销婚姻**：胁迫；隐瞒重大疾病

### 夫妻关系

**夫妻共同财产**（婚姻关系存续期间所得）：
1. 工资、奖金、劳务报酬
2. 生产、经营、投资收益
3. 知识产权收益
4. 继承/受赠财产（确定只归一方除外）

**夫妻一方财产**：婚前财产；人身损害赔偿/补偿；遗嘱/赠与合同中确定只归一方的财产

### 离婚

**离婚冷静期**：自婚姻登记机关收到离婚登记申请之日起**30日内**，任何一方不愿离婚的，可撤回申请；30日期满后双方应亲自到婚姻登记机关申请发给离婚证，未申请视为撤回。

### 收养

- 被收养人：**未满14周岁**的未成年人
- 收养人条件：无子女或只有一名子女；有抚养、教育和保护能力；未患有医学上认为不应当收养子女的疾病；年满30周岁
- 收养人收养子女数：**一般无子女者最多2名，有子女者最多1名**

## 第六编 继承

### 法定继承

**继承顺序**：

| 顺序 | 继承人 | 范围 |
|------|--------|------|
| 第一顺序 | 配偶、子女、父母 | 丧偶儿媳/女婿尽了主要赡养义务的，作为第一顺序继承人 |
| 第二顺序 | 兄弟姐妹、祖父母、外祖父母 | - |

> 继承开始后，由第一顺序继承人继承，第二顺序不继承。无第一顺序继承人继承的，由第二顺序继承人继承。

**遗产分配**：
- 同一顺序继承人**一般应当均等**
- 对生活有特殊困难又缺乏劳动能力的继承人，分配时应予照顾
- 对被继承人尽了主要扶养义务或与被继承人共同生活的继承人，分配时可以多分
- 有扶养能力和有扶养条件的继承人，不尽扶养义务的，分配时应不分或少分

### 遗嘱继承和遗赠

**遗嘱形式**：公证遗嘱、自书遗嘱、代书遗嘱、打印遗嘱、录音录像遗嘱、口头遗嘱

**遗嘱效力**：公证遗嘱效力最高（民法典施行后已取消公证遗嘱效力优先规则，遗嘱时间先后为主要依据）

**遗赠扶养协议**：扶养人承担生养死葬义务，享受遗赠权利。遗赠扶养协议优先于遗嘱和法定继承。

### 遗产处理

- 遗产分割应当有利于生产和生活需要，不损害遗产的效用
- 遗产分割时应保留胎儿的继承份额
- 无人继承又无人受遗赠的遗产：归国家所有；死者生前是集体所有制组织成员的，归所在集体所有制组织所有

## 第七编 侵权责任

### 一般规定

**归责原则**：以**过错责任**为原则，以过错推定、无过错责任为补充

**损害赔偿范围**：医疗费、护理费、交通费、营养费、住院伙食补助费、误工费；造成残疾的：辅助器具费和残疾赔偿金；造成死亡的：丧葬费和死亡赔偿金

**精神损害赔偿**：侵害自然人人身权益造成严重精神损害的，被侵权人有权请求精神损害赔偿

### 产品责任（第一千二百零二条至第一千零六十七条）

- 生产者承担**无过错责任**
- 销售者承担**过错责任**（产品缺陷由生产者造成时，销售者赔偿后可向生产者追偿）
- 因产品缺陷危及他人人身、财产安全的，被侵权人有权请求生产者、销售者承担**排除妨碍、消除危险、停止侵害**等侵权责任
- **惩罚性赔偿**：明知产品存在缺陷仍然生产、销售，或未及时采取补救措施，造成死亡或健康严重损害的，被侵权人有权请求相应的**惩罚性赔偿**

### 交通事故责任（第一千二百零八条至第一千二百三十三条）

- 机动车发生交通事故造成损害的，依照**道路交通安全法**的有关规定承担赔偿责任
- 机动车交通事故责任适用**过错责任**原则
- 租赁、借用机动车发生事故：车辆所有人对损害有过错的，承担相应赔偿责任
- 未经允许驾驶他人机动车：驾驶人承担赔偿责任，所有人、管理人对有过错的，承担相应责任
- 交通事故中行人故意碰撞机动车的，机动车一方不承担赔偿责任

### 医疗损害责任（第一千二百一十八条至第一千二百二十八条）

- 医务人员在诊疗活动中未尽到与当时的医疗水平相应的诊疗义务，造成患者损害的，医疗机构承担**赔偿责任**
- 需要实施手术、特殊检查、特殊治疗的：医务人员应当及时向患者说明医疗风险、替代医疗方案等情况，并取得其明确同意
- 因抢救生命垂危的患者等紧急情况，不能取得患者或其近亲属意见的，经医疗机构负责人批准，可以立即实施相应的医疗措施
- **举证责任**：患者在诊疗活动中受到损害，医疗机构或者医务人员有过错的，由医疗机构承担赔偿责任

### 环境污染和破坏生态责任（第一千二百二十九条至第一千二百三十五条）

- 污染环境、破坏生态造成他人损害的，侵权人承担**无过错责任**
- 两个以上侵权人共同造成损害的，承担**连带责任**
- 侵权人故意污染环境、破坏生态造成严重后果的，被侵权人有权请求相应的**惩罚性赔偿**

### 高度危险责任（第一千二百三十六条至第一千二百四十四条）

- 从事高度危险作业造成他人损害的，承担**无过错责任**
- 民用核设施或核材料事故：营运人承担无过错责任（最高赔偿额由国务院制定）
- 民用航空器造成他人损害：经营者承担无过错责任

### 饲养动物损害责任（第一千二百四十五条至第一千二百五十一条）

- 饲养的动物造成他人损害的，动物饲养人或管理人承担**无过错责任**
- 违反管理规定，未对动物采取安全措施造成损害的，**同样适用无过错责任**
- 动物园的动物造成损害：动物园承担**过错推定责任**（能证明已尽到管理职责的，不承担责任）
- 遗弃、逃逸的动物造成损害：由原动物饲养人或管理人承担侵权责任

### 建筑物和物件损害责任（第一千二百五十二条至第一千二百五十八条）

- 建筑物、构筑物或其他设施倒塌造成他人损害：建设单位和施工单位承担**连带责任**
- 建筑物、构筑物或其他设施搁置物、悬挂物脱落、坠落造成损害：使用人/管理人承担**过错推定责任**
- 堆放物倒塌、滚落或滑落造成损害：堆放人承担过错责任（不能证明自己没有过错的，承担相应责任）
- 林木折断、树木果实坠落造成损害：树木所有人或管理人承担过错责任
- 地面施工、窨井等地下工作物造成损害：施工人承担过错推定责任
