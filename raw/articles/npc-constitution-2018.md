## 序言（核心要点）

### 历史定位
- 中国是世界上历史最悠久的国家之一，各族人民共同创造了光辉灿烂的文化
- 一八四〇年后中国变为半殖民地、半封建国家
- 一九四九年新中国建立，人民成为国家的主人
- 社会主义制度已确立，剥削制度已消灭

### 指导思想
> 在马克思列宁主义、毛泽东思想、邓小平理论、"三个代表"重要思想、科学发展观、**习近平新时代中国特色社会主义思想**指引下

### 根本任务
- 沿着**中国特色社会主义道路**
- 集中力量进行社会主义现代化建设
- 推动物质文明、政治文明、精神文明、社会文明、**生态文明**协调发展
- 建设富强民主文明和谐美丽的**社会主义现代化强国**
- 实现**中华民族伟大复兴**

### 基本制度
- **人民民主专政**（实质上即无产阶级专政）
- **社会主义道路**
- **改革开放**
- **社会主义市场经济**
- **社会主义法治**
- **新发展理念**

### 重要政治制度
> 在长期的革命、建设、改革过程中，已经结成由中国共产党领导的，有各民主党派和各人民团体参加的，包括全体社会主义劳动者、社会主义事业的建设者、拥护社会主义的爱国者、拥护祖国统一和致力于中华民族伟大复兴的爱国者的广泛的爱国统一战线

- **中国共产党领导的多党合作和政治协商制度**将长期存在和发展
- **台湾**是中华人民共和国神圣领土的一部分
- 中国坚持和平发展道路，推动构建**人类命运共同体**

### 宪法地位
> 本宪法以法律的形式确认了中国各族人民奋斗的成果，规定了国家的根本制度和根本任务，是国家的根本法，**具有最高的法律效力**

## 第一章 总纲

### 国体与根本制度

| 条款 | 内容 |
|------|------|
| **第一条** | 中华人民共和国是**工人阶级领导的、以工农联盟为基础的人民民主专政的社会主义国家**。社会主义制度是根本制度。**中国共产党领导是中国特色社会主义最本质的特征**。 |
| **第二条** | 中华人民共和国一切权力**属于人民**。人民行使国家权力的机关是全国人大和地方各级人大。 |
| **第三条** | 国家机构实行**民主集中制**原则。 |

### 民族政策

> 第四条：中华人民共和国各民族**一律平等**。各少数民族聚居的地方实行**区域自治**，设立自治机关，行使自治权。各民族都有使用和发展自己的语言文字的自由。

### 法治建设

> 第五条：中华人民共和国实行**依法治国，建设社会主义法治国家**。一切法律、行政法规和地方性法规都不得同宪法相抵触。任何组织或者个人都不得有超越宪法和法律的特权。

### 经济制度

| 条款 | 内容 |
|------|------|
| **第六条** | 社会主义经济制度的基础是**生产资料的社会主义公有制**（全民所有制和劳动群众集体所有制）。坚持**公有制为主体、多种所有制经济共同发展**的基本经济制度。 |
| **第七条** | **国有经济**是国民经济中的主导力量 |
| **第八条** | 农村集体经济组织实行**家庭承包经营为基础、统分结合的双层经营体制** |
| **第十条** | 城市的土地属于**国家所有**；农村和城市郊区的土地除法律规定属于国家所有的外属于**集体所有** |
| **第十一条** | **个体经济、私营经济等非公有制经济**是社会主义市场经济的重要组成部分 |
| **第十八条** | 允许外国企业和其他经济组织或个人依照中国法律在中国投资 |

### 社会权利与保障

| 条款 | 内容 |
|------|------|
| **第十三条** | 公民的**合法的私有财产不受侵犯** |
| **第十四条** | 国家建立健全**同经济发展水平相适应的社会保障制度** |
| **第二十四条** | 国家**倡导社会主义核心价值观**：爱祖国、爱人民、爱劳动、爱科学、爱社会主义 |
| **第二十五条** | 国家推行**计划生育** |
| **第二十六条** | 国家保护和改善生活环境和生态环境，防治污染和其他公害 |

### 行政区划

> 第三十条：全国分为**省、自治区、直辖市**；省、自治区分为**自治州、县、自治县、市**；县、自治县分为**乡、民族乡、镇**。

> 第三十一条：国家在必要时得设立**特别行政区**。

### 国家工作人员

> 第二十七条：国家工作人员就职时应当依照法律规定公开进行**宪法宣誓**。

## 第二章 公民的基本权利和义务

### 平等权与人权

> 第三十三条：凡具有中华人民共和国国籍的人都是中华人民共和国公民。中华人民共和国公民在**法律面前一律平等**。国家**尊重和保障人权**。

### 政治权利

| 条款 | 内容 |
|------|------|
| **第三十四条** | 年满十八周岁公民有**选举权和被选举权**（依法被剥夺政治权利者除外） |
| **第三十五条** | 有**言论、出版、集会、结社、游行、示威**的自由 |
| **第三十六条** | 有**宗教信仰自由**。宗教团体和宗教事务不受外国势力的支配。 |
| **第四十一条** | 对国家机关和国家工作人员有**批评、建议、申诉、控告、检举**的权利 |

### 人身自由

| 条款 | 内容 |
|------|------|
| **第三十七条** | 人身自由不受侵犯。禁止非法拘禁和以其他方法非法剥夺或者限制公民的人身自由。 |
| **第三十八条** | **人格尊严**不受侵犯 |
| **第三十九条** | **住宅**不受侵犯 |
| **第四十条** | **通信自由和通信秘密**受法律保护 |

### 社会经济权利

| 条款 | 内容 |
|------|------|
| **第四十二条** | 有**劳动的权利和义务** |
| **第四十三条** | 劳动者有**休息**的权利 |
| **第四十四条** | 国家实行**退休制度** |
| **第四十五条** | 在年老、疾病或丧失劳动能力时有从国家和社会获得**物质帮助**的权利 |
| **第四十六条** | 有**受教育**的权利和义务 |
| **第四十七条** | 有进行**科学研究、文学艺术创作和其他文化活动**的自由 |

### 特定群体保护

| 条款 | 内容 |
|------|------|
| **第四十八条** | 妇女在各方面享有同男子**平等**的权利，实行**男女同工同酬** |
| **第四十九条** | 婚姻、家庭、母亲和儿童受国家保护 |
| **第五十条** | 保护**华侨、归侨、侨眷**的合法权利和利益 |

### 公民义务

| 条款 | 内容 |
|------|------|
| **第五十二条** | 维护**国家统一**和全国各民族**团结** |
| **第五十三条** | 遵守宪法和法律，保守**国家秘密**，爱护公共财产，遵守劳动纪律、公共秩序，尊重社会公德 |
| **第五十四条** | 维护祖国的**安全、荣誉和利益** |
| **第五十五条** | **保卫祖国、抵抗侵略**是神圣职责；依照法律**服兵役**和参加民兵组织是光荣义务 |
| **第五十六条** | 有依照法律**纳税**的义务 |

> 第五十一条：公民在行使自由和权利的时候，**不得损害国家的、社会的、集体的利益**和其他公民的合法的自由和权利。

## 第三章 国家机构

### 第一节 全国人民代表大会

#### 地位与组成
- **最高国家权力机关**，常设机关是全国人大常委会
- 行使**国家立法权**
- 代表由省、自治区、直辖市、特别行政区和军队选出，各少数民族都应当有适当名额的代表
- 每届任期**五年**

#### 重要职权

| 条款 | 职权内容 |
|------|----------|
| **第六十二条** | 修改宪法；监督宪法实施；制定和修改基本法律；选举国家主席、副主席；决定国务院总理人选；选举中央军委主席；选举国家监察委员会主任；选举最高法院院长；最高检检察长；审查批准计划和预算；批准省自治区直辖市建置；决定特别行政区的设立及制度；决定战争和和平的问题等 |
| **第六十三条** | **罢免**主席、副主席、总理、军委主席、监察委员会主任、最高法院院长、最高检检察长 |
| **第六十四条** | 宪法修改须全国人大常委会或**五分之一以上**代表提议，由全国人大**三分之二以上**多数通过 |

#### 全国人大常委会

| 条款 | 内容 |
|------|------|
| **组成** | 委员长、副委员长若干人、秘书长、委员若干人 |
| **任期** | 同全国人大每届任期相同；委员长、副委员长连续任职**不得超过两届** |
| **重要职权** | 解释宪法、监督宪法实施；制定和修改除基本法律以外的其他法律；解释法律；监督国务院、军委、监察委员会、最高法院、最高检的工作；决定特赦；决定战争状态的宣布；决定全国或局部动员；决定全国或个别省市区进入紧急状态等（21项职权） |

#### 代表权利保障
> 第七十四条：全国人大代表非经主席团许可，不受逮捕或者刑事审判。

> 第七十五条：全国人大代表在全国人大各种会议上的发言和表决，**不受法律追究**。

### 第二节 中华人民共和国主席

| 条款 | 内容 |
|------|------|
| **任职条件** | 年满**四十五周岁**的中华人民共和国公民 |
| **任期** | 同全国人大每届任期相同 |
| **主要职权** | 根据人大及常委会决定公布法律；任免国务院总理等；授予勋章和荣誉称号；发布特赦令、紧急状态令、战争动员令；代表国家进行国事活动，接受外国使节；派遣和召回驻外全权代表；批准和废除条约 |
| **继任规定** | 主席缺位时由副主席继任；副主席缺位时由人大补选；都缺位时由人大常委会委员长暂时代理 |

| 条款 | 内容 |
|------|------|
| **地位** | **中央人民政府**，最高国家权力机关的执行机关，最高国家行政机关 |

| **主要职权** | 规定行政措施、制定行政法规、发布决定和命令；统一领导各部和委员会的工作；编制和执行计划及预算；领导和管理全国行政工作；改变或撤销各部委及地方各级行政机关的不适当的决定、命令；批准省自治区直辖市的区域划分，批准自治州、县、自治县、市的建置和区域划分等 |

### 第八节 人民法院和人民检察院

#### 人民法院（审判机关）

| 层级 | 名称 |
|------|------|
| 最高 | **最高人民法院** |
| 高级 | 各省、自治区、直辖市高级人民法院 |
| 中级 | 各省、自治区按地区设立的中级人民法院；直辖市设置的中级人民法院；自治州中级人民法院等 |
| 基层** | 各市、市辖区、县、自治县人民法院 |

> 第一百二十八条：最高人民法院对**全国人民代表大会和全国人大常委会**负责。地方各级人民法院对产生它的国家权力机关负责。

#### 人民检察院（法律监督机关）

| 层级 | 名称 |
|------|------|
| 最高 | **最高人民检察院** |
| 地方 | 地方各级人民检察院 |
| 专门 | 军事检察院等专门人民检察院 |

> 第一百三十七条：最高人民检察院对**全国人民代表大会和全国人大常委会**负责。地方各级人民检察院对产生它的国家权力机关和上级人民检察院负责。

## 关键特点总结

1. **宪法最高法律效力**：一切法律、行政法规都不得同宪法相抵触
2. **四项基本权利**：选举权与被选举权、言论出版集会结社游行示威自由、宗教信仰自由、人身自由不受侵犯
3. **公民义务**：维护国家统一、依法纳税、服兵役、遵守宪法和法律
4. **国家机构体系**：民主集中制原则，全国人大为最高权力机关，国务院为最高行政机关
5. **2018年修正新增**：包括"习近平新时代中国特色社会主义思想"、"人类命运共同体"、"监察委员会"等
