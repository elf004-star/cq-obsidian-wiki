# 反不正当竞争法

## 概述

**《中华人民共和国反不正当竞争法》**是我国维护市场竞争秩序的基本法律。1993年首次通过，2017年第一次修订，2019年修正，**2025年6月第二次修订**（2025年10月15日施行）。共5章43条。

## 核心原则

> 经营者在生产、经营活动中，应当遵循：**自愿、平等、公平、诚信**的原则，遵守法律和商业道德，公平参与市场竞争。

## 九类不正当竞争行为

| 类型 | 条款 | 核心内容 |
|------|------|----------|
| **混淆行为** | 第七条 | 仿冒他人商品名称、包装、装潢、姓名、域名、网站名称等 |
| **商业贿赂** | 第八条 | 禁止向交易相对方工作人员、利用职权者行贿 |
| **虚假宣传** | 第九条 | 禁止虚假商业宣传及组织虚假交易、虚假评价 |
| **侵犯商业秘密** | 第十条 | 定义明确，举证责任倒置（权利人提供初步证据后由涉嫌侵权人举证） |
| **有奖销售禁止** | 第十一条 | 抽奖式有奖销售最高奖不得超过**5万元** |
| **商业诋毁** | 第十二条 | 禁止编造、传播虚假信息损害竞争对手商誉 |
| **网络不正当竞争** | 第十三条 | 互联网专条：强制跳转、恶意不兼容、数据侵权、虚假交易等 |
| **强制交易** | 第十四条 | 禁止平台强制经营者低于成本价销售 |
| **优势地位滥用** | 第十五条 | 禁止大企业滥用优势拖欠中小企业账款（2025年新增） |

## 商业秘密保护要点

**定义：** 不为公众所知悉、具有商业价值并经权利人采取相应保密措施的技术信息、经营信息等商业信息。

**举证责任：** 权利人提供初步证据后，涉嫌侵权人应当证明其不存在侵犯商业秘密的行为。

**赔偿：** 故意侵犯商业秘密（情节严重）可获 **1-5倍** 惩罚性赔偿。

## 互联网专条（第十三条）

2025年修订扩充后的互联网专条涵盖：

- 利用数据、算法、技术、平台规则实施的不正当竞争
- 强制插入链接、强制跳转
- 误导、欺骗、强迫用户修改、关闭、卸载其他经营者的产品或服务
- 恶意不兼容
- 以不正当方式获取、使用其他经营者的数据
- 滥用平台规则实施虚假交易、虚假评价或恶意退货

## 平台责任（第二十一条）

平台经营者须：
1. 在平台服务协议和交易规则中**明确平台内公平竞争规则**
2. 建立**不正当竞争举报投诉和纠纷处置机制**
3. 发现平台内经营者实施不正当竞争行为，应**及时依法采取必要处置措施**
4. 保存有关记录，并向**平台经营者住所地县级以上**监督检查部门报告

## 法律责任概览

| 责任类型 | 关键数字 |
|----------|----------|
| 法定赔偿上限 | **500万元** |
| 惩罚性赔偿 | 侵犯商业秘密情节严重者 **1-5倍** |
| 罚款下限 | 妨碍调查（个人1万以下，单位10万以下） |

## 域外适用

本法适用于在**中华人民共和国境外**实施的不正当竞争行为，只要其扰乱境内市场竞争秩序，损害境内经营者或消费者合法权益。
