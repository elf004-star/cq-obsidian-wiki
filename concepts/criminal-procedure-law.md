# 刑事诉讼法

## 概述

**《中华人民共和国刑事诉讼法》**是我国程序法核心，规范刑事案件从立案到执行的全流程。1979年7月首次通过，1996年、2012年、2018年三次修正。全文共5编、15章、290条余。

核心任务（第二条）：保证准确、及时查明犯罪事实，正确应用法律，惩罚犯罪分子，**保障无罪的人不受刑事追究**，尊重和保障人权。

## 五编结构

| 编 | 内容 | 核心章节 |
|----|------|----------|
| **第一编** | 总则 | 任务和基本原则、管辖、回避、辩护与代理、证据、强制措施 |
| **第二编** | 立案、侦查和提起公诉 | 侦查（讯问/询问/搜查/技术侦查）、公诉 |
| **第三编** | 审判 | 一审（含简易程序/速裁程序）、二审、死刑复核、再审 |
| **第四编** | 执行 | 判决/裁定的执行程序 |
| **第五编** | **特别程序** | 未成年人、和解公诉案件、缺席审判、违法所得没收、精神病人强制医疗 |

## 公检法分工（第三条）

| 机关 | 职责 |
|------|------|
| **公安机关** | 侦查、拘留、执行逮捕、预审 |
| **人民检察院** | 批准逮捕、自侦案件侦查、提起公诉 |
| **人民法院** | 审判（独立行使审判权） |
| **国家安全机关** | 办理危害国家安全刑事案件（行使与公安机关相同职权） |

## 特别程序要点

### 未成年人刑事案件诉讼程序
- **附条件不起诉**：检察院可对未成年人附条件不起诉
- 犯罪记录封存：符合条件的未成年人犯罪记录应予封存

### 当事人和解的公诉案件诉讼程序
- 适用范围：因民间纠纷引起的轻微犯罪案件、过失犯罪案件
- 效果：和解协议可作为从宽处理依据

### 缺席审判程序
- 适用范围：犯罪嫌疑人/被告人在境外，监察机关/公安机关已掌握犯罪事实及证据的案件

### 违法所得没收程序
- 犯罪嫌疑人、被告人逃匿、死亡时，可依法没收其违法所得

### 精神病人强制医疗程序
- 经法定程序鉴定不负刑事责任的精神病人，可强制医疗

## 与刑法的关系

- [[criminal-law|刑法]] 规定**什么行为构成犯罪**（实体法）
- 刑事诉讼法规定**如何追究犯罪**（程序法）
- 两法配套：刑法是实体依据，刑诉法是程序保障
