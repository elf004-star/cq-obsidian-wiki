# 中华人民共和国著作权法

## 基本信息

- **首次通过**：1990年9月7日
- **施行日期**：1991年6月1日
- **历史沿革**：
  - 2001年10月27日第一次修正
  - 2010年2月26日第二次修正
  - **2020年11月11日第三次修正**（现行版本）
- **施行日期（现行）**：2021年6月1日

## 著作权内容

### 人身权（保护期不受限制）

| 权利 | 定义 |
|------|------|
| 发表权 | 决定作品是否公之于众 |
| 署名权 | 表明作者身份，在作品上署名 |
| 修改权 | 修改或授权他人修改作品 |
| 保护作品完整权 | 保护作品不受歪曲、篡改 |

### 财产权（可许可、转让，保护期为作者终生及死后50年）

包括：复制权、发行权、出租权、展览权、表演权、放映权、广播权、信息网络传播权、摄制权、改编权、翻译权、汇编权

## 权利归属规则

| 情形 | 归属规则 |
|------|----------|
| 一般规则 | 著作权属于作者 |
| 法人作品 | 由法人或非法人组织主持、代表其意志创作并承担责任 |
| 演绎作品 | 著作权由演绎人享有（不得侵犯原作品著作权） |
| 合作作品 | 著作权由合作作者共同享有 |
| 职务作品 | 一般归作者享有；特殊职务作品由单位享有（作者保留署名权） |
| 委托作品 | 合同约定；未约定或无合同的，属于受托人 |

## 法定许可（可不经许可但需支付报酬）

包括：个人学习研究引用、新闻报道引用、课堂教学少量复制、馆藏复制、免费表演、少数民族文字翻译、无障碍提供等

## 侵权赔偿标准（第54条）

| 计算方式 | 赔偿数额 |
|----------|----------|
| 首选 | 权利人实际损失或侵权人违法所得 |
| 次选 | 参照权利使用费 |
| 故意侵权且情节严重 | 在上述数额1倍以上5倍以下 |
| 难以计算 | 人民法院判决500元至500万元 |
