# 公民的基本权利和义务

## 权利分类

### 政治权利

| 条款 | 权利内容 |
|------|----------|
| 第三十四条 | 选举权和被选举权（年满18周岁） |
| 第三十五条 | 言论、出版、集会、结社、游行、示威的自由 |
| 第三十六条 | 宗教信仰自由 |
| 第四十一条 | 批评、建议、申诉、控告、检举权 |

### 人身自由

| 条款 | 权利内容 |
|------|----------|
| 第三十七条 | 人身自由不受侵犯 |
| 第三十八条 | 人格尊严不受侵犯 |
| 第三十九条 | 住宅不受侵犯 |
| 第四十条 | 通信自由和通信秘密受保护 |

### 社会经济权利

| 条款 | 权利内容 |
|------|----------|
| 第四十二条 | 劳动的权利和义务 |
| 第四十三条 | 休息权 |
| 第四十五条 | 物质帮助权（年老、疾病、丧失劳动能力） |
| 第四十六条 | 受教育权 |
| 第四十七条 | 科研、文艺创作自由 |

### 特定群体保护

| 条款 | 保护对象 |
|------|----------|
| 第四十八条 | 妇女平等权 |
| 第四十九条 | 婚姻、家庭、母亲和儿童 |
| 第五十条 | 华侨、归侨、侨眷 |

## 公民义务

| 条款 | 义务内容 |
|------|----------|
| 第五十二条 | 维护国家统一和民族团结 |
| 第五十三条 | 遵守宪法法律、保守国家秘密、爱护公共财产 |
| 第五十四条 | 维护国家安全、荣誉和利益 |
| 第五十五条 | 保卫祖国、服兵役 |
| 第五十六条 | 依法纳税 |

## 行使权利的限制

> 第五十一条：公民在行使自由和权利的时候，**不得损害国家的、社会的、集体的利益**和其他公民的合法的自由和权利。
