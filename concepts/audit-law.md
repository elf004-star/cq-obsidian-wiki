# 审计法

## 概述

**《中华人民共和国审计法》**是我国审计监督的基本法律。1994年8月31日首次通过，2006年2月28日第一次修正，**2021年10月23日第二次修正**（现行版本）。共**7章68条**。

## 核心原则

> **坚持中国共产党对审计工作的领导**，构建**集中统一、全面覆盖、权威高效**的审计监督体系。

**四项基本原则：** 独立、客观、公正、保密。

## 审计机关体系

| 层级 | 设置 | 领导关系 |
|------|------|----------|
| **审计署** | 国务院（总理领导） | 主管全国审计工作 |
| **省级审计机关** | 省级人民政府 | 本级政府负责 + 上级审计机关领导（业务为主） |
| **市/县级审计机关** | 市/县人民政府 | 本级政府负责 + 上级审计机关领导（业务为主） |

> **独立性保障：** 审计机关依法独立行使审计监督权，不受其他行政机关、社会团体和个人的干涉。审计机关负责人没有违法失职的，不得随意撤换。

## 审计范围

| 审计对象 | 审计内容 | 主要依据条款 |
|----------|----------|-------------|
| **政府预算执行** | 本级各部门、下级政府预算执行和决算 | 第18条 |
| **国家事业组织** | 财务收支 | 第21条 |
| **国有企业、国有金融机构** | 资产、负债、损益及其他财务收支 | 第22条 |
| **政府投资项目** | 预算执行、决算及重大公共工程项目 | 第23条 |
| **国有资源、国有资产** | 管理使用情况 | 第24条 |
| **社会保险基金、全国社会保障基金** | 财务收支 | 第24条 |
| **国际组织/外国政府援助贷款项目** | 财务收支 | 第25条 |

## 审计权限体系

### 资料获取权
- 有权要求被审计单位提供**财务会计资料**、业务管理资料、电子数据
- 国家政务信息系统和数据共享平台按规定向审计机关开放

### 检查权
- 检查财务会计资料、实物资产、信息系统的**安全性、可靠性、经济性**

### 调查权
- 经批准可查询被审计单位在**金融机构**的账户

### 强制措施
- **封存**有关资料和违规资产
- 通知**暂停拨付或暂停使用**与违规行为直接有关的款项

## 审计程序

| 阶段 | 要点 |
|------|------|
| **审计通知** | 实施审计**3日前**送达审计通知书；特殊情况经批准可持通知书直接审计 |
| **审计实施** | 审查资料、检查实物、向有关单位和个人调查（不少于2人） |
| **征求意见** | 审计报告报送审计机关前征求被审计单位意见，被审计单位**10日内**提出书面意见 |
| **出具报告** | 审计机关出具审计报告，依法作出审计决定 |

## 整改与问责

**整改要求：**
- 被审计单位按规定时间整改
- 将整改情况报告审计机关
- 同时向本级人民政府或有关主管机关、单位报告
- 按照规定向社会公布

> **结果运用：** 审计结果及整改情况应当作为**考核、任免、奖惩领导干部**和制定政策、完善制度的重要参考。

## 救济途径

| 审计决定类型 | 救济途径 |
|--------------|----------|
| **财务收支**审计决定 | 可申请**行政复议**或提起**行政诉讼** |
| **财政收支**审计决定 | 提请审计机关的本级人民政府裁决（裁决为最终决定） |

## 特别审计类型

| 类型 | 说明 |
|------|------|
| **经济责任审计** | 对领导干部任期经济责任履行情况进行审计 |
| **自然资源资产离任审计** | 对领导干部任职期间自然资源资产管理和生态环境保护责任进行审计 |
