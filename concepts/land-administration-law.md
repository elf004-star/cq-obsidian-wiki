# 土地管理法

## 概述

**《中华人民共和国土地管理法》**是我国土地管理的基本法律。1986年首次通过，历经3次修正（1988、2004、2019年）。2019年修正版为现行版本，共**8章87条**。2020年1月1日起施行。

## 核心原则

> **十分珍惜、合理利用土地和切实保护耕地是我国的基本国策。**

## 土地公有制

| 类型 | 说明 |
|------|------|
| **全民所有制（国有）** | 城市市区的土地属于国家所有；由国务院代表国家行使所有权 |
| **劳动群众集体所有制** | 农村和城市郊区的土地（除法定国有）属于农民集体所有；宅基地和自留地、自留山属于农民集体所有 |

## 土地分类体系

| 类别 | 包含类型 |
|------|----------|
| **农用地** | 耕地、林地、草地、农田水利用地、养殖水面等 |
| **建设用地** | 城乡住宅、公共设施、工矿、交通水利、旅游、军事设施用地等 |
| **未利用地** | 农用地和建设用地以外的土地 |

## 耕地保护制度

### 占补平衡原则
"**占多少，垦多少**" — 建设用地必须占用多少耕地，就开垦同等数量和质量的耕地。

### 永久基本农田

| 要点 | 内容 |
|------|------|
| 划定范围 | 粮棉油糖生产基地、高标准农田、蔬菜基地、农业科研试验田 |
| 比例要求 | 各省一般应占本行政区域内耕地的**80%以上** |
| 占用审批 | 必须经**国务院**批准 |
| 禁止行为 | 发展林果业、挖塘养鱼、建窑建坟、采矿取土等 |

## 土地征收制度

### 征地审批权限

| 情形 | 审批机关 |
|------|---------|
| 永久基本农田 | **国务院** |
| 永久基本农田外耕地超35公顷 | **国务院** |
| 其他土地超70公顷 | **国务院** |
| 其他土地 | 省、自治区、直辖市人民政府 |

### 征地补偿原则

> 征收土地应当给予**公平、合理的补偿**，保障被征地农民原有生活水平不降低、长远生计有保障。

### 征地程序

1. 现状调查 + **社会稳定风险评估**
2. 公告至少**30日**
3. 多数成员认为方案不符规定 → **召开听证会**
4. 签订补偿安置协议
5. 申请征收土地

## 宅基地制度

| 要点 | 内容 |
|------|------|
| 一户一宅 | 一户只能拥有一处宅基地 |
| 面积标准 | 由省、自治区、直辖市规定 |
| 限制审批 | 出卖、出租、赠与住宅后，再申请宅基地不予批准 |
| 自愿退出 | 国家允许进城落户村民**依法自愿有偿退出**宅基地 |

## 集体经营性建设用地入市（2019年修正亮点）

- 符合规划、依法登记的集体经营性建设用地可**出让、出租**
- 须经本集体经济组织成员村民会议**三分之二以上**成员同意
- 取得的使用权可**转让、互换、出资、赠与或抵押**
- **不再必须先征为国有**

## 土地承包期限

| 土地类型 | 承包期 | 届满政策 |
|----------|--------|----------|
| 耕地 | **30年** | 再延长**30年** |
| 草地 | **30-50年** | 届满依法相应延长 |
| 林地 | **30-70年** | 届满依法相应延长 |

## 法律责任要点

| 违法行为 | 处罚 |
|----------|------|
| 非法占用耕地 | 退还土地、限期拆除或没收建筑物、罚款 |
| 闲置耕地满1年 | 缴纳闲置费；满2年可无偿收回 |
| 非法批准用地 | 批准文件无效、处分责任人 |
| 侵占挪用征地补偿费 | 追刑责或处分 |
