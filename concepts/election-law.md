# 选举法

## 概述

**《中华人民共和国全国人民代表大会和地方各级人民代表大会选举法》**是我国规范各级人大代表选举的基本法律。1979年7月1日第五届全国人大第二次会议通过，1980年1月1日起施行，历经**5次修正**（1982、1986、1995、2004、2010年）。共12章57条。

## 核心原则

| 原则 | 内容 |
|------|------|
| **普遍性** | 年满18周岁公民都有选举权（依法被剥夺政治权利者除外） |
| **平等性** | 每一选民在一次选举中只有一个投票权 |
| **直接选举与间接选举结合** | 县级以下人大代表直接选举；县级以上人大代表间接选举 |
| **秘密投票** | 一律采用无记名投票方法 |

## 选举方式

| 选举层级 | 选举方式 |
|----------|----------|
| 全国人大代表 | **间接选举**（由下一级人大选举） |
| 省、自治区、直辖市、设区的市、自治州人大代表 | **间接选举** |
| 不设区的市、市辖区、县、自治县、乡、民族乡、镇人大代表 | **直接选举** |

## 代表名额分配

### 各级人大代表名额上限

| 行政层级 | 名额上限 |
|----------|----------|
| 全国人大 | **不超过3000人** |
| 省、自治区、直辖市 | 1000名 |
| 设区的市、自治州 | 650名（人口超1000万时） |
| 不设区的市、市辖区、县、自治县 | 450名（人口超165万时） |
| 乡、民族乡、镇 | 160名（人口不足2000可少于40名） |

### 少数民族代表名额

| 聚居少数民族占总人口比例 | 代表名额比例 |
|--------------------------|--------------|
| 30%以上 | 与当地代表名额相同 |
| 15%-30% | 不超过总名额的30% |
| 不足15% | 可少于1/2（特少民族至少1人） |

## 选举程序要点

### 时间节点

| 程序 | 时间要求 |
|------|----------|
| 公布选民名单 | 选举日**20日以前** |
| 公布初步候选人名单 | 选举日**15日以前** |
| 公布正式候选人名单 | 选举日**7日以前** |
| 停止候选人介绍 | 选举日 |

### 差额选举

| 选举类型 | 差额比例 |
|----------|----------|
| 直接选举 | 多于应选名额**1/3至1倍** |
| 间接选举 | 多于应选名额**1/5至1/2** |

### 投票与当选

| 条件 | 直接选举 | 间接选举 |
|------|----------|----------|
| 选举有效 | 全体选民**过半数**参加投票 | 全体代表**过半数**参加 |
| 当选条件 | 获得参加投票选民**过半数**选票 | 获得全体代表**过半数**选票 |

## 代表的监督、罢免、辞职

| 程序 | 要件 |
|------|------|
| 县级人大代表罢免 | 原选区选民**50人以上**联名 |
| 乡级人大代表罢免 | 原选区选民**30人以上**联名 |
| 罢免通过 | 原选区**过半数**选民通过（县级、乡级） |
