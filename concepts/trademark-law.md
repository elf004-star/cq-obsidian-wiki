# 中华人民共和国商标法

## 基本信息

- **首次通过**：1982年8月23日
- **最新修正**：2019年4月23日（第四次修正）
- **施行日期**：1983年3月1日（初始）；现行版本持续施行
- **主管机关**：国家知识产权局商标局（注册和管理）、商标评审委员会（争议处理）

## 核心制度

### 商标类型（第3条）

| 类型 | 定义 |
|------|------|
| 商品商标 | 用于商品上的普通商标 |
| 服务商标 | 用于服务上的普通商标 |
| 集体商标 | 供团体、协会成员使用，表明成员资格的标志 |
| 证明商标 | 由具有监督能力的组织控制，证明商品原产地、原料等特定品质 |

### 禁止注册绝对理由（第10条）

**八类绝对禁止标志**：
1. 同中国国家名称、国旗、国徽等相同或近似
2. 同外国国家名称、国旗、国徽等相同或近似（经该国政府同意的除外）
3. 同政府间国际组织名称、旗帜、徽记等相同或近似
4. 与官方标志、检验印记相同或近似（经授权的除外）
5. 同"红十字"、"红新月"的名称、标志相同或近似
6. 带有**民族歧视性**的
7. 带有**欺骗性**，使公众对商品质量等产生误认的
8. **有害于社会主义道德风尚**或有其他不良影响的

### 禁止注册相对理由（第13-16条）

- **代理人抢注禁止**（第15条）：未经授权，代理人或代表人不得注册被代理人/被代表人的商标
- **在先权利保护**（第32条）：不得损害他人现有的在先权利，不得以不正当手段抢先注册他人已经使用并有一定影响的商标

### 驰名商标保护（第13-14条）

| 情形 | 保护措施 |
|------|----------|
| 相同/类似商品上复制、摹仿或翻译他人**未在中国注册**的驰名商标 | 不予注册并禁止使用 |
| 不同/不相类似商品上复制、摹仿或翻译他人**已在中国注册**的驰名商标 | 不予注册并禁止使用 |

**认定因素**：相关公众知晓程度、持续使用时间、宣传覆盖面、保护的记录等

### 异议程序（第33条）

- 初步审定公告之日起**三个月内**可提出异议
- 在先权利人、利害关系人可提异议的事由：驰名商标保护、代理人抢注、地理标志、相同近似商标等
- 任何人可提异议的事由：恶意注册、禁用标志、缺乏显著特征等

### 续展制度（第40条）

| 项目 | 规定 |
|------|------|
| 续展时限 | 期满前**十二个月**内按规办理 |
| 宽展期 | 期满后**六个月** |
| 续展有效期 | 每次**十年** |
| 未续展后果 | 注销注册商标 |
