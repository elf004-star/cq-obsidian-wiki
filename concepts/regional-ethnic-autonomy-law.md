# 民族区域自治法

## 概述

**《中华人民共和国民族区域自治法》**是我国第一部关于民族区域自治的基本法律。1984年5月31日第六届全国人大第二次会议通过并公布，1984年10月1日起施行。2001年2月28日第九届全国人大常委会第二十次会议修正。是国家的一项**基本政治制度**。

## 核心原则

> 中华人民共和国是全国各族人民共同缔造的**统一的多民族国家**。民族区域自治是中国共产党运用马克思列宁主义解决我国民族问题的**基本政策**，是国家的一项**基本政治制度**。

**三大原则：**
1. 国家充分尊重和保障各少数民族管理本民族内部事务权利
2. 坚持各民族平等、团结
3. 各民族共同繁荣

## 自治地方体系

| 层级 | 数量 | 名称示例 |
|------|------|----------|
| **自治区** | 5个 | 内蒙古自治区、广西壮族自治区、宁夏回族自治区、新疆维吾尔自治区、西藏自治区 |
| **自治州** | 30个 | 延边朝鲜族自治州、甘孜藏族自治州等 |
| **自治县** | 120+个 | 鄂温克族自治旗、土家族苗族自治州等 |

**稳定性保障：** 民族自治地方一经建立，未经法定程序，不得撤销或者合并；区域界线一经确定，未经法定程序，不得变动。

## 自治机关

| 机关 | 人员要求 |
|------|----------|
| 人民代表大会 | 实行区域自治的民族的代表 + 其他居住在本行政区域内的民族的适当名额代表 |
| 人大常委会 | 应当有实行区域自治的民族的公民担任主任或副主任 |
| 人民政府 | **自治区主席、自治州州长、自治县县长由实行区域自治的民族的公民担任** |

## 自治权要点

| 类别 | 主要权利 |
|------|----------|
| **立法自治权** | 自治区报全国人大常委会批准；自治州、自治县报省级人大常委会批准 |
| **变通执行权** | 上级国家机关的决议、决定如不适合本地实际情况，可报批准后变通执行或停止执行（60日内答复） |
| **财政自治权** | 一级财政，自主安排使用财政收入；享受财政转移支付制度照顾；机动资金、预备费比例高于一般地区 |
| **经济自主权** | 自主安排经济建设事业；合理调整经济结构；自然资源优先合理开发利用 |
| **语言文字权** | 使用和发展本民族语言文字；以实行区域自治的民族的语言文字为主 |
| **教育自治权** | 自主决定教育规划、学校设置、办学形式；双语教育 |
| **文化自治权** | 自主发展民族文化事业；保护历史文化遗产 |
| **资源管理权** | 草原、森林确定所有权和使用权；禁止毁草毁林开垦耕地 |

## 与选举法的关系

民族区域自治地方的**人民代表大会代表名额**由选举法规定：

| 聚居少数民族占总人口比例 | 代表名额比例 |
|--------------------------|--------------|
| 30%以上 | 与当地代表名额相同 |
| 15%-30% | 不超过总名额的30% |
| 不足15% | 可少于1/2（特少民族至少1人） |
