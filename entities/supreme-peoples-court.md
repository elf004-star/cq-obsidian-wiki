# 最高人民法院

## 概述

**最高人民法院**是中华人民共和国最高审判机关，对全国人民代表大会和全国人民代表大会常务委员会负责。

## 法院体系（四级两审制）

| 层级 | 法院名称 |
|------|----------|
| 最高 | 最高人民法院 |
| 高级 | 各省、自治区、直辖市高级人民法院 |
| 中级 | 地区中级人民法院；直辖市中级人民法院；自治州中级人民法院 |
| 基层 | 市、市辖区、县、自治县人民法院 |

## 主要职能

- 审理各类重大案件
- 监督地方各级人民法院和专门人民法院的审判工作
- 统一法律适用标准
- 死刑复核（2021年以前由最高人民法院统一行使）
