# 全国人民代表大会

## 概述

**全国人民代表大会**（简称"全国人大"）是中华人民共和国最高国家权力机关，常设机关是全国人民代表大会常务委员会。

## 地位

- **最高国家权力机关**
- 行使**国家立法权**
- 在国家机构体系中居于首位

## 组成与任期

| 项目 | 内容 |
|------|------|
| 代表产生 | 由省、自治区、直辖市、特别行政区和军队选出 |
| 少数民族 | 各少数民族都应当有适当名额的代表 |
| 任期 | **每届五年** |

## 重要职权

| 条款 | 职权内容 |
|------|----------|
| 第六十二条 | 修改宪法；监督宪法实施；制定和修改基本法律；选举国家主席、副主席；决定国务院总理人选；选举中央军委主席；选举国家监察委员会主任；选举最高法院院长；最高检检察长；审查批准计划和预算；批准省自治区直辖市建置；决定特别行政区的设立及制度；决定战争和和平的问题等 |
| 第六十三条 | 罢免主席、副主席、总理、军委主席、监察委员会主任、最高法院院长、最高检检察长 |
| 第六十四条 | 宪法修改须全国人大常委会或五分之一以上代表提议，由全国人大三分之二以上多数通过 |

## 全国人大常委会

| 项目 | 内容 |
|------|------|
| 组成 | 委员长、副委员长若干人、秘书长、委员若干人 |
| 任期 | 同全国人大每届任期相同 |
| 委员长、副委员长 | 连续任职**不得超过两届** |
| 职权 | 解释宪法、监督宪法实施；制定和修改除基本法律以外的其他法律；解释法律；监督"一府两院"工作；决定特赦、战争状态、紧急状态等（21项职权） |

## 代表权利保障

| 条款 | 保障内容 |
|------|----------|
| 第七十四条 | 全国人大代表非经主席团许可，**不受逮捕或刑事审判** |
| 第七十五条 | 全国人大代表在全国人大各种会议上的发言和表决，**不受法律追究** |
