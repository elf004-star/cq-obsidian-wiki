# 国务院

## 概述

**国务院**即中华人民共和国中央人民政府，是最高国家权力机关的执行机关，也是最高国家行政机关。

## 基本信息

| 项目 | 内容 |
|------|------|
| 地位 | 中央人民政府；最高国家行政机关；最高国家权力机关的执行机关 |
| 组成 | 总理、副总理若干人、国务委员若干人、各部部长、各委员会主任、审计长、秘书长 |
| 领导体制 | **总理负责制**；各部、各委员会实行部长、主任负责制 |
| 任期 | 同全国人大每届任期相同；总理、副总理、国务委员连续任职**不得超过两届** |
| 组织法 | 《国务院组织法》（1982年初施，**2024年3月修订**） |

## 国务院组织法（2024年修订）

**《国务院组织法》**于1982年12月10日首次通过，2024年3月11日第十四届全国人大第二次会议修订。这是该法42年来首次修订，主要变化包括：

### 新增条款

| 条款 | 主要内容 |
|------|----------|
| **党的建设**（第三条） | 明确国务院坚持党的领导，贯彻习近平新时代中国特色社会主义思想，建设法治政府、创新政府、廉洁政府和服务型政府 |
| **全体会议制度**（第七条） | 明确国务院全体会议和常务会议制度 |
| **重大决策程序**（第十六条） | 健全科学决策、民主决策、依法决策制度，规范重大行政决策程序 |
| **行政监督**（第十七条） | 加强行政复议、备案审查、行政执法监督、政府督查，坚持政务公开 |

### 组成与机构

| 机构类型 | 说明 |
|----------|------|
| 组成部门（各部、委、行、署） | 设部长（主任、行长、审计长）1人，副部长2-4人 |
| 直属机构 | 主管专门业务，由国务院任免 |
| 办事机构 | 协助总理办理专门事项 |

## 主要职权

- 规定行政措施、制定行政法规、发布决定和命令
- 统一领导各部和委员会的工作
- 编制和执行计划及预算
- 领导和管理全国行政工作
- 改变或撤销各部委及地方各级行政机关的不适当的决定、命令
- 批准省自治区直辖市的区域划分，批准自治州、县、自治县、市的建置和区域划分
