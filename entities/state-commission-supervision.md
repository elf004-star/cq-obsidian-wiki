# 国家监察委员会

## 概述

**国家监察委员会**是中华人民共和国最高国家监察机关，与中央纪律检查委员会合署办公，实行一套工作机构、两块牌子。

## 历史沿革

2018年3月，第十三届全国人大一次会议通过《中华人民共和国宪法修正案》，在国家机构中增设"监察委员会"，同年颁布《监察法》，完成国家监察体制改革。

## 基本信息

| 项目 | 内容 |
|------|------|
| 设立依据 | 2018年宪法修正案、《中华人民共和国监察法》 |
| 领导体制 | 主任负责制 |
| 与纪委关系 | 合署办公（中纪委为国家监察委员会的上级机关） |
| 监督对象 | 所有行使公权力的公职人员（"监察全覆盖"） |

## 主要职能

- **监督**：对公职人员依法履职、秉公用权、廉洁从政情况进行监督
- **调查**：对涉嫌贪污贿赂、滥用职权、玩忽职守等职务违法和职务犯罪进行调查
- **处置**：对违法的公职人员作出政务处分决定；对涉嫌职务犯罪的移送检察机关依法起诉

## 与检察院的关系（2018年后）

| 对比维度 | 国家监察委员会 | 人民检察院 |
|----------|---------------|-----------|
| 性质 | 国家监察机关 | 国家检察机关 |
| 核心职能 | 监督 + 调查（职务违法/犯罪） | 起诉 + 法律监督 |
| 2018年前 | 无（2018年新设） | 职务犯罪侦查权 |
| 2018年后 | 承接检察院转隶的职务犯罪侦查权 | 保留公诉、法律监督职能 |

2018年后，检察院不再行使职务犯罪侦查权，相关职能整体转隶至国家监察委员会。两者在反腐败工作中形成"监察调查—检察起诉"的工作衔接机制。
